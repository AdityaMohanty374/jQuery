HTML is the backbone of the web. It defines the structure and content of a page using elements like headings, paragraphs, links, images, and forms.

CSS controls the visual presentation of HTML elements — colors, typography, spacing, layout, and animations. It separates design from content.

JavaScript adds interactivity and logic to the browser. It can manipulate the DOM, respond to user events, fetch data from APIs, and much more.

jQuery is a fast, lightweight library that wraps browser APIs into a simple, expressive syntax — making DOM selection, events, and AJAX much easier to write.
